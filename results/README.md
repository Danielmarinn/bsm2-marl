# Results

This folder contains lightweight, portfolio-friendly result assets generated from the tracked example training log.

## Current contents

- `ctrl2_qint_training_overview.png`: summary figure for the current CTRL-2 SAC run.
- `ctrl2_qint_summary.md`: short write-up of the ongoing result.
- `generate_ctrl2_qint_results.py`: reproducible script used to generate the figure and summary from `logs/ctrl2_qint_training.csv`.

## Notes

- These are ongoing results, not final thesis benchmarks.
- The goal is to show real project progress and system behavior without overstating conclusions.
