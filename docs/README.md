# Docs Guide

## Main files
- `correlation-analysis/bsm2_corr_analysis_full.m`: current main correlation-analysis script kept at the cleaner canonical path.
- `correlation-analysis/bsm2_corr_analysis_full_prog_RL_snapshot.m`: older imported snapshot from the `prog_RL` workspace, kept so no analysis work is lost.
- `correlation-analysis/open-loop-vs-closed-loop-correlation.md`: note explaining why variable selection used closed-loop data.
- `matlab-python-integration-notes.txt`: notes about the MATLAB-Python CSV bridge and signal mapping.

## Notes
- The repo keeps both correlation scripts because they are not identical.
- Prefer the canonical `correlation-analysis/` folder for future work.
- Keep new notes and analysis material in ASCII-friendly filenames when possible.
- See the root `README.md` for the project overview and `PROJECT_STRUCTURE.md` for the folder map.
